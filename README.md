## 🚀 Getting Started

Follow these steps to run locally:

```bash
# 1. Clone the repository
git clone https://github.com/Dhruv0551/Netfix-clone.git
cd Netfix-clone

# 2. Install dependencies
npm install

# 3. Start the development server
npm start
```
